PictureSelectUploadView -> PictureSelectBottomDialogView -> PictureSelectCompressView -> PictureSelectOriginView
PictureSelectOriginView: 最原始的相册选择功能
PictureSelectCompressView: 具有压缩的相册选择功能
PictureSelectBottomDialogView: 支持底部选择的相册选择功能
PictureSelectUploadView: 支持自动上传文件的相册选择功能

如果只想要 选择+压缩+自动上传, 不需要底部选择呢?
